plotMutFreqPDF will plot the number of mutations per sequences as a
histogram. 

  [Gx, Ax] = plotMutFreqPDF

  [Gx, Ax] = plotMutFreqPDF(MutData)

  [Gx, Ax] = plotMutFreqPDF(VDJdata,VDJheader)

  [Gx, Ax] = plotMutFreqPDF(...,Param,Value,...)
 
  [Gx, Ax] = plotMutFreqPDF(...,P)

  P = plotMutFreqPDF('getinput')

  INPUT
    MutData: structured data of pairwise mutations, returned by
      getMutFreqData.
    VDJdata: MxN cell of all VDJ annotation data
    VDJheader: 1xN cell of all header name for VDJdata
    P: a structure containing P.(Param) = Value information. This input
       option might be easier for adjusting plot parameters. You can also
       use the direct param, cell pair input listed below. To make a
       default P structure, use P = plotMutFreqPDF('GetInput');

    Some plot features can be set using the following Param-Value pairs:
      Param           Value       Description
      --------------  ----------  --------------------------------------
      PairWith        'parent'    Uses SHM data between child and parent
                      'germline'  Uses SHM data between child and germline
      AddLegend       'y' 'n'     Show legend
      FontName        Arial       Font name of X and Y axes
      FontSize        10          Font size of X and Y axes
      FigHeight       inches      Height of the whole figure
      FigWidth        inches      Width of the whole figure
      LegendFontSize  12          Set the font size of the legend
      MaxBars         15          Set how many bars to plot
      Edges           Nx1 mat     Left edges of histogram

    *To save plot, use Param-Value pairs in the savePlot.m function.

  OUTPUT
    Gx: figure handle
    Ax: axes handle
    P: Structure of inputs, returned from plotMutFreqPDF('getinput')

  See also getMutFreqData, savePlot
