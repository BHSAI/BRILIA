filterFastqByQuality will go through a fastq file and trim sequences based
on the quality threshold specified by the user, and then return the
trimmed sequences as a fastq file. Duplicates sequences are joined BEFORE
trimming by quality. 

  INPUT
    InputFile: file name of the input fastq
    OutputFile: file name of the ouput fastq (will append '.fastq' if this
      name does not end with it.
    MinQuality: minimum quality for reads to be used for trimming
