dispList will display all contents of a string cell, listing it by number
too. Used mainly for interfacing with the user what option to select.
  
  dispList(A)

  EXAMPLE 
    A = {'What'; 'Is'; 'This'; 1322 ; 2332; 31423}
    dispList(A)
         1) What
         2) Is
         3) This
         4) 1322
         5) 2332
         6) 31423
